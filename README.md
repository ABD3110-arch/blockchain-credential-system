# 🎓 Blockchain Academic Credential System

## 📌 Overview

This project is a blockchain-based system for secure and forgery-resistant academic credential verification.

## 🚀 Features

* Tamper-proof credential storage
* Admin-controlled issuance
* IPFS-based file storage
* Instant verification

## 🛠 Tech Stack

* Solidity
* Ganache
* Remix IDE
* MetaMask
* IPFS (Pinata)
* Ethers.js

## ⚙️ How It Works

1. Upload certificate to IPFS
2. Store hash on blockchain
3. Verify using smart contract

## 📂 Files

* AcademicCredentials.sol → Smart contract
* index.html → Frontend interface

## 🔐 Security

* Role-based access control
* Duplicate prevention
* Immutable records
