Setup Guide
1

Install Ganache

Download Ganache from trufflesuite.com/ganache. Start a new workspace — it runs a local Ethereum chain at http://127.0.0.1:7545 with 10 pre-funded accounts.

2

Deploy via Remix IDE

Open remix.ethereum.org. Paste the AcademicCredentials.sol contract, set compiler to 0.8.17, then in the Deploy tab select Injected Provider - MetaMask or Web3 Provider pointed to Ganache. Deploy and copy the contract address.

3

Configure MetaMask

Add a custom network in MetaMask: RPC URL http://127.0.0.1:7545, Chain ID 1337, Symbol ETH. Import a Ganache private key as the admin account.

4

Update Contract Address Below

Paste your deployed contract address into the field below. The ABI is already embedded in this page.

5

Upload Documents to IPFS

Use web3.storage or pinata.cloud to upload degree certificate PDFs. Copy the Qm... CID hash and use it as the IPFS Hash when issuing credentials.
