How Verification Works

1

Hash Computation

A SHA-256 hash is computed from the credential's JSON data using deterministic sorting.

2

On-Chain Lookup

The hash is queried against the Ethereum smart contract registry — immutable and tamper-proof.

3

Result

If the hash matches an active credential, verification passes. Any modification to the data will fail.
